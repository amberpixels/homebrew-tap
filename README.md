# amberpixels/homebrew-tap

Homebrew tap for [amberpixels](https://amberpixels.io) command-line tools.

## Usage

```sh
brew tap amberpixels/tap
brew install <tool>
```

Or in one shot, without tapping first:

```sh
brew install amberpixels/tap/<tool>
```

## Available tools

| Tool | Description |
|------|-------------|
| [`ele`](https://github.com/amberpixels/ele) | Quiet the elephant — a drop-in `pg_restore` wrapper with live progress and a classified error summary. |

## Notes

Formulae and casks in this tap are generated automatically by
[GoReleaser](https://goreleaser.com) on each upstream release — don't edit them
by hand.
